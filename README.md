# pets_n_pals
This is a self educational project utilizing popular dependencies such as: 

• Redux-ToolKit
• Express
• Formik
• React-Router-Dom
• axios

and more 
 
## Getting Started

```bash
cd client 
npm install
cd server 
npm install
```

First, run the development server:

```bash
npm run dev
# or
yarn dev
```
